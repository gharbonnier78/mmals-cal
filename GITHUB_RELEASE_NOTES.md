# MMALS-CAL v0.3.2 - Cross-Benchmark Evidence Synthesis

A submission and reproducibility release for regime-aware conformal evidence verification over seven continual-learning traces.

### Included
- complete replay code;
- seven portable filtered traces;
- compiled arXiv-style article;
- deterministic no-leakage split;
- five calibration strategies;
- protocol and claim verification;
- engineering and 12-year-old explanations.

### Important boundary
This is an offline final-checkpoint replay, not an online change detector or runtime adaptive calibration controller.
