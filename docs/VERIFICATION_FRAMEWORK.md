# Verification Framework

MMALS-CAL uses four complementary layers:

1. **Metric Evaluator:** What happened numerically?
2. **Protocol Verifier:** Was the experimental evidence generated without forbidden leakage or incompatible units?
3. **MMALS-CAL:** Is the uncertainty evidence calibrated, regime-appropriate, compact, and actionable?
4. **Claim Verifier:** Which scientific statements are supported, unsupported, or prohibited?

This is empirical and statistical evidence verification. It is not formal verification of all possible runtime behaviors.
